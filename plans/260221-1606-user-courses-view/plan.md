## Overview
Cho phép Quản trị viên (Admin) xem danh sách các Khóa học/Môn học mà một Người dùng cụ thể đang sở hữu (bao gồm cả trạng thái khóa học) ngay tại bảng Manage Users để dễ dàng hỗ trợ và kiểm soát.

## Tech Stack
- Frontend: Next.js + Tailwind + shadcn/ui Dialog
- Backend: Node.js (Express) + MongoDB
- Logic: Thêm Route mới cho Student tìm qua User Email hoặc ID.
